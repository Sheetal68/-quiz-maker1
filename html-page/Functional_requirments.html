About Quizzer

Quizzer is a web application,that Make it easy for Instructor to create his online quiz easily and Students Can enroll into quiz , this will help instructor get accurate quiz result.

Functional Requirments

At the next Few lines we listed some of Functional Requirments that we hope our project Can have:

  • Every Instructor MUST Register as an Instructor and login into our website in order to create a quiz.
  • Every Student MUST Register as a Student and login into our website in order to enroll into his quiz.
  • Only Instructor can create quiz.
  • Instructor has an ability to delete quiz,Updete quiz questions.
  • Student can enroll only ONE time into quiz.
  • Instructor can view the grades of students who enroled into the quiz.
  • Instructor can set a TIMER for quiz.
  • Students can enroll into course via a generated access code.
  • Instructor can add some resources that a student can learn from.
